Configurar almacenamiento
Para guardar los datos online necesitas un GitHub Token. Solo se configura una vez y queda guardado en tu navegador.
📋 Cómo obtener tu token:
1. Ve a github.com → Settings → Developer settings
2. Clic en Personal access tokens → Tokens (classic)
3. Generate new token (classic)
4. Escribe un nombre (ej: "compras"), marca solo el permiso gist
5. Clic en Generate token y cópialo aquí
🔒 Tu token se guarda solo en este navegador, nunca se envía a ningún servidor externo salvo GitHub.
